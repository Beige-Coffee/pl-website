# Course Guide

Before we dive deeper, let's take a quick tour of what's available to you throughout this course.

## Your Toolbox

Click the **TOOLS** dropdown in the top-right corner of the tutorial to access your toolbox:

- **Bitcoin Node**: A terminal connected to your own private **Bitcoin regtest node**. You'll use this to simulate broadcasting transactions, mining blocks, and ultimately opening Lightning channels. Click the **?** button for a list of available commands. **Tip:** Press **Tab** to autocomplete command names (press Tab repeatedly to cycle through matches).
- **Scratchpad**: A sandbox for running snippets of Python code to help you learn. Want to print the value of a variable or experiment with a function? Each exercise has a **"Send to Sandbox"** button that lets you send your code and data to the Scratchpad so you can explore freely.
- **Transactions**: A notebook that stores your Lightning transactions as you build them and advance through your payment channel.
- **Exercise Files**: A file browser that lets you view and revisit the code you've written across exercises.

As you read, you'll also encounter **checkpoint quizzes** (short knowledge-check questions inline in the text), **coding exercises** (an editor will appear where you write Python that runs directly in the browser), and **TX generators** (interactive tools that build real Bitcoin transactions from your exercise solutions).
